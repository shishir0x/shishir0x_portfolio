## Editor

- My primary editor is **Antigravity** for intelligent, agentic coding.
- I am currently using the **Dark Modern** theme to keep things sleek and easy on the eyes.

## Terminal

- **[Warp](https://www.warp.dev/)** is my go-to terminal for its speed and AI-assisted workflows.
- **[Windows Terminal](https://www.microsoft.com/en-us/p/windows-terminal/9n0dx20hk701)** when I need a native Windows command line experience.

## Desktop Apps

- **[Tableau](https://www.tableau.com/)** and **[Power BI](https://powerbi.microsoft.com/)** for building interactive dashboards and data storytelling.
- **[Jupyter Notebooks](https://jupyter.org/)** for Python data analysis, EDA, and model building.
- **[DBeaver](https://dbeaver.io/)** as my universal database tool for writing SQL and managing connections.
- **[Microsoft Excel](https://www.microsoft.com/en-us/microsoft-365/excel)** (because you can never truly escape Excel as an analyst).
- **[Brave](https://brave.com/)** is my main browser for research and web tools.
- **[Slack](https://slack.com/)** and **[Discord](https://discord.com/)** for team communication.
- **[Spotify](https://www.spotify.com/)** for deep-work focus playlists.
